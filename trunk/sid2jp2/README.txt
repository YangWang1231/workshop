sid2jp 0.5 (Beta)
---------------------------------------------------------------

1. Install Visual C++ 8.0 libraries

vcredist_x86.exe

2. Install latest FWTools

http://fwtools.maptools.org/

3. Copy sid2jp2.exe file to C:\Program Files\FWToolsX.Y.Z\bin

where X.Y.Z is version number of FWTools.

4. Run sid2jp2.exe
